Pulling Changes

After changes have been made and committed in one repository, you can pull these changes into a local repository.

To pull changes from another repository:

  1. In the sidebar select the local repository which will receive the pulled changes (the Destination repository).

  2. Click the Pull icon in the top bar of the Document Window, or select the menu item Repository > Pull from...

  3. In the dialog that appears, select a compatible repository from the drop down menu under source.

  4. If needed, choose any advanced options which are appropriate when pulling (for example, the --force option if the pull will create remote heads).
