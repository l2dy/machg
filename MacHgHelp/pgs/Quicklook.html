Using QuickLook

You can use Quick Look, in all the main views of MacHg: The Browser View, The History View, and The Differences View. The shortcut to display the quicklook preview of the selected items is the standard OSX shortcut of the space bar.

Important:If you want to copy or select text from the QuickLook preview panel then you can double click on the previewed file to be taken to the snapshot of that file. The historical snapshots of the various files are stored in your Caches folder and are erased when you quit MacHg.

To use QuickLook in the Browser View:

  1. If not already in the Browser view, switch to it (by selecting Browser View from the Document View menu).

  2. Select the files in the browser you want to preview.

  3. Open the QuickLook panel by any of:

    • Choose the menu item File > Open Quick Look Panel.

    • Choose the QuickLook toolbar item if you have included it in the tool bar.

    • Use the space bar shortcut on your keyboard.

To use QuickLook in the History View:

  1. If not already in the History View, switch to it (by selecting History View from the Document View menu).

  2. Select the revisions in the History View for which you want to view snapshots of the files as of those revisions. If you select multiple revisions the preview of the files which will be shown will be the version of the file as of the most recent revision selected.

  3. Open the QuickLook panel by any of:

    • Choose the menu item File > Open Quick Look Panel.

    • Choose the QuickLook toolbar item if you have included it in the tool bar.

    • Use the space bar shortcut on your keyboard.

To use QuickLook in the Differences View:

  1. If not already in the Differences View, switch to it (by selecting Differences View from the Document View menu).

  2. Select the base revision and the compare revision in the Differences view.

  3. The changes between the revisions are represented in virtual files listed in the browser part of the view. Amongst these files choose the files you want to preview. The preview will show the state of the files at the time of the compare revision.

  4. Open the QuickLook panel by any of:

    • Choose the menu item File > Open Quick Look Panel.

    • Choose the QuickLook toolbar item if you have included it in the tool bar.

    • Use the space bar shortcut on your keyboard.
